## Merge Sorted Arrays

Takes 3 pre-sorted arrays and combines them into 1 ascending sorted array:
- `collection_1`: max → min (descending)
- `collection_2`: min → max (ascending)
- `collection_3`: min → max (ascending)

**Returns**: Single sorted array (min → max ascending)

## Tech Stack

- **TypeScript** - Type-safe JavaScript
- **Jest** - Unit testing framework
- **ts-node** - Run TypeScript
- **Node.js** - Runtime

## How to run

```bash
# 1. Install dependencies
pnpm install

# 2. Run example
pnpm start

# 3. Run tests
pnpm test

```
